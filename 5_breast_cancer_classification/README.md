# 유방암 환자 분류 모델링

## 1. 사전 세팅

---

## 2. 문제 정의

* 유방암 의심 환자 데이터를 활용해 **악성(M)과 양성(B)을 분류하는 모델**을 구축한다.
* 의료 도메인 해석보다는 **모델링 기법 비교를 중심으로 정확한 분류 모델 탐색**에 초점을 둔다.

---

## 3. 데이터 간단 확인

* 총 569행 × 33열, 결측치 없음
* 불필요한 컬럼(`Unnamed: 32`) 제거 후 32개 Feature 사용
* 타겟 변수: `diagnosis` (M: 악성, B: 양성)

---

## 4. 문제 해결 프로세스 정의

* **문제:** 영상에서 추출한 수치형 지표를 이용해 암의 양성/악성 여부를 예측
* **기대 효과:** 의사의 판단을 보조하고, 악성 환자 누락(FN)을 최소화
* **해결 방안:** Logistic, KNN, LDA, SVM, RandomForest 등 이진 분류 모델 비교
* **성과 측정:** Accuracy, Precision, Recall, F1, ROC-AUC 등 다각적 평가

---

## 5. 데이터 분리

* 타겟: `diagnosis` → M=1, B=0
* 학습 70% / 테스트 30%, `stratify=y`로 클래스 비율 유지

---

## 6. Data Readiness Check

* `Unnamed: 32` 제거, `diagnosis` 매핑 완료
* 수치형 Feature 표준화(StandardScaler) 적용
* 이상치 및 분포 확인 완료 (EDA)

---

## 7. Feature Engineering

* 표준화로 스케일 차이 제거
* 상관계수 히트맵으로 Feature 간 다중공선성 점검

---

## 8. 모델링 및 평가

| Model                   | Accuracy | Recall (악성) | 특징                 |
| :---------------------- | :------: | :---------: | :----------------- |
| **Logistic Regression** | **0.96** |   **0.92**  | 단순 모델이지만 가장 안정적    |
| LDA                     |   0.93   |     0.84    | 경계 학습 기반, 비교적 빠름   |
| SVM                     |   0.95   |     0.89    | RBF 커널, 비선형 패턴 반영  |
| KNN                     |   0.96   |     0.92    | 거리 기반, 데이터 작을 때 유리 |
| RandomForest            |   0.94   |     0.88    | 시간이 오래 걸려 적합하지 않음       |

---

## 9. 결론

* **Logistic Regression**이 가장 높은 정확도(0.9649)와 재현율(0.9219)을 기록
* 데이터의 선형성이 강해 단순 모델이 오히려 적합
* **트리 기반 모델은 해석력 확보용**으로만 유용했으며, 결과적으로 “도메인 지식이 없어도 모델링으로 데이터 패턴을 이해할 수 있음”을 검증
